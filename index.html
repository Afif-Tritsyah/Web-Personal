  • Sublime Text

    Sublime Text is a premium code editor known for its incredible speed and responsiveness.   • PhpMyAdmin

    phpMyAdmin is a PHP-based web application used to manage MySQL databases through a browser. This tool provides a graphical interface for creating, editing, and deleting databases, tables, and data. Developers can run SQL queries, import/export databases, and perform database administration without the need for the MySQL command line.

  • XAMPP

    XAMPP is a software package that includes the Apache web server, MySQL database, PHP, and Perl in a single installation package. This tool allows developers to run a local development environment on their computers without having to configure each component separately. XAMPP is very practical for testing and developing PHP web applications before deploying them to production servers.

  • Figma

    Figma is a cloud-based collaborative design platform for creating UI/UX designs, wireframes, and prototypes. This tool allows multiple designers to work in real time on the same project, providing a component library, design system, and interactive prototyping features. Figma also facilitates design handoff to developers with an inspect feature that displays CSS code and exports assets.
